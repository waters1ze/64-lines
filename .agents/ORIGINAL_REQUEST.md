Мы создаем функционал шахматных задач (Puzzles), приватных видеоуроков, удаление чатов и UI-улучшений для образовательной платформы "Шахматная школа 64 линии".

Working directory: d:/Шахматная школа 64 линии

## Requirements

### R1. Puzzles Feature
В компоненте `teacher-hub.tsx` создать новый раздел "Задачи" (Puzzles), в котором ученик может решать тактические задачи. При открытии раздела отправляется запрос на `/api/puzzles` для получения случайной задачи подходящего рейтинга (или с RapidAPI, или из базы `Puzzle`, логика в API уже есть). Пользователь решает задачу на доске `react-chessboard`. Если ход правильный, отправляется POST-запрос на `/api/puzzles/submit` с `isCorrect: true`, иначе `false`. Если сервер возвращает `error: 'LIMIT_REACHED'`, доска блокируется и появляется сообщение о необходимости Premium-подписки для безлимита.

### R2. Premium Video Library
В компоненте `teacher-hub.tsx` (в разделе `VideoLibrary`) добавить визуальное отображение "замка" для видео с флагом `isPremium`, если `isPremium` у пользователя (пропсы) равен `false`. Клик по премиум-видео должен вызывать функцию покупки, вместо открытия видео. Для админа добавить чекбокс "Premium" при создании видео (потребуется обновить API создания видео).

### R3. Chat History Management
В `ChatComponent` добавить кнопку "Очистить чат", вызывающую новый эндпоинт `/api/chat/delete` (который нужно реализовать), полностью удаляющий сообщения между текущим пользователем и собеседником. Также создать эндпоинт `/api/cron/cleanup-messages` для удаления сообщений старше 48 часов.

### R4. Premium Leaderboard Aesthetics
В разделе `Leaderboard` (таблица лидеров) добавить визуальное выделение пользователей с `isPremium`. Строки этих пользователей должны иметь золотистый фон (например, `bg-yellow-500/10`) и/или границу.

## Acceptance Criteria

### Puzzles
- [ ] Раздел "Задачи" корректно загружает FEN-позицию и реагирует на ходы пользователя через `chess.js`.
- [ ] При лимите (LIMIT_REACHED) доска блокируется.

### Videos
- [ ] Не-Premium пользователи не могут открыть премиум-видео.
- [ ] Админы могут ставить галочку "Premium" при добавлении видео.

### Chat
- [ ] Кнопка "Очистить чат" успешно стирает текущий диалог из БД через API.
