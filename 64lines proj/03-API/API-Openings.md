# API Дебютных Библиотек
> Управляет базой шахматных дебютов (PGN файлов), доступных для тренировок и изучения теории.

## Расположение в коде
`app/api/openings/route.ts, app/api/openings/[id]/route.ts`

## Детали
- **GET `/api/openings`**: Возвращает список всех доступных дебютных материалов.
- **POST `/api/openings`**: Создает новый дебютный PGN файл в базе данных.
- **PUT/DELETE `/api/openings/[id]`**: Редактирование PGN/названия или удаление дебюта.

## Связи
- Использует: [[Model-Opening]]
- Используется в: [[teacher-hub]] (раздел Openings)

## Заметки / особенности
Используется для загрузки эталонных дебютных древ (например, Сицилианская защита) для разбора на уроках.
