# API Личного Профиля
> Позволяет авторизованному пользователю получить детальную информацию о своем профиле из базы данных или изменить личные данные.

## Расположение в коде
`app/api/profile/route.ts`

## Детали
- **GET `/api/profile`**: Возвращает полную запись текущего пользователя [[Model-User]] (рейтинг, streak, премиум статус, привязанный учитель).
- **POST `/api/profile`**: Изменение имени пользователя или пароля. Требует передачи старого пароля для подтверждения при смене пароля.

## Связи
- Использует: [[Model-User]]
- Используется в: [[teacher-hub]] (SettingsPanel)

## Заметки / особенности
Для предотвращения устаревания сессии NextAuth на клиенте, главная страница делает прямой запрос к БД и переопределяет JWT-данные.
