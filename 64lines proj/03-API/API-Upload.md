# API Загрузки Файлов
> Вспомогательный роут для загрузки медиафайлов, PGN баз и PDF документов во внутреннее облачное хранилище или локальную папку.

## Расположение в коде
`app/api/upload/route.ts`

## Детали
- **POST `/api/upload`**:
  - Принимает `multipart/form-data` файл.
  - Сохраняет файл и возвращает публичный URL для сохранения ссылки в моделях БД.

## Связи
- Использует: нет прямых связей с БД
- Используется в: [[teacher-hub]] (при создании курсов, уроков и видео)

## Заметки / особенности
Обычно настроен на работу с Vercel Blob или локальным диском в зависимости от среды окружения.
