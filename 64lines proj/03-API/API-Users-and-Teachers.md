# API Управления Пользователями
> Предоставляет администраторам функции модерации пользователей базы данных, а ученикам — возможность просмотра списка тренеров.

## Расположение в коде
`app/api/users/route.ts, app/api/users/teachers/route.ts`

## Детали
1. **GET `/api/users`**:
   - Возвращает список всех пользователей системы с пагинацией и поиском. Требует роль `ADMIN`.
2. **POST `/api/users`**:
   - Редактирование роли пользователя (ADMIN, TEACHER, STUDENT) или удаление пользователя.
3. **GET `/api/users/teachers`**:
   - Выводит список всех зарегистрированных тренеров школы (используется учениками в чате и поиске).

- Используется в: [[teacher-hub]] (раздел UsersManager, ChatComponent)

Роут защищен строгой серверной проверкой роли сессии.
