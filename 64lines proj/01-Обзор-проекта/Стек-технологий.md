# Стек технологий проекта "64 линии"

Проект построен на современном веб-стеке с использованием серверного рендеринга и API на базе Next.js.

## 🛠️ Основные технологии

- **Фреймворк**: **Next.js 14 / 16** (App Router). Практически вся логика страниц и API эндпоинтов сосредоточена в каталоге `app/`.
- **Язык разработки**: **TypeScript**. Строгая типизация моделей данных, пропсов компонентов и API-контрактов.
- **База данных**: **PostgreSQL** — надежная реляционная СУБД для хранения учетных записей, чатов, задач и логов.
- **ORM (Слой работы с БД)**: **Prisma Client**. Схема описана в `prisma/schema.prisma`. Генерирует клиентские типы для работы с данными на сервере.
- **Аутентификация**: **NextAuth.js** (Credentials Provider). Сессии хранятся в зашифрованном JWT. Роль пользователя проверяется при каждом запросе к API.
- **Стилизация**: **Tailwind CSS v4** + **shadcn/ui** (компоненты на базе Radix UI). Обеспечивают темную/светлую тему, адаптивную сетку и высокую скорость верстки.
- **Иконки**: **Lucide React**.

## ♟️ Шахматная логика и UI

- **chess.js**: Библиотека для валидации ходов по правилам шахмат, генерации легальных ходов, парсинга PGN и FEN позиций.
- **react-chessboard**: React-компонент для рендеринга шахматной доски с поддержкой Drag & Drop фигур, кастомизацией клеток и разметкой.
- **@uiw/react-md-editor**: Редактор Markdown с превью для написания комментариев к урокам, анализу партий и описаниям курсов тренерами.

## 📡 Интеграции и Сервисы

- **YooMoney Quickpay API (Эквайринг)**: Вебхуки с SHA-1 подписью для автоматической активации покупок и Premium подписки.
- **Jitsi Meet API (Iframe API)**: Встраивание видеоконференций прямо в веб-интерфейс живого урока без необходимости установки приложений.
- **nodemailer (SMTP-клиент)**: Отправка email-сообщений для верификации почты и восстановления паролей (использует `lib/mail.ts`).
