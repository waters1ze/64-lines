# Переменные окружения и Конфигурация

Файлы конфигурации и окружения определяют режимы работы шахматной школы на локальной странице и на продакшн-сервере.

---

## 🔐 Переменные окружения (`.env`)

Файл `.env` содержит чувствительные ключи и настройки подключения. Пример шаблона (`.env.example`):

| Переменная | Описание / Значение | Используется в |
|---|---|---|
| `DATABASE_URL` | Строка подключения к PostgreSQL database | Prisma Client, `lib/db.ts` |
| `NEXTAUTH_SECRET` | Ключ шифрования сессий JWT NextAuth | NextAuth, `app/api/auth/[...nextauth]` |
| `NEXTAUTH_URL` | Полный URL сайта (например, `http://localhost:3000`) | NextAuth, коллбеки |
| `YOOMONEY_SECRET` | Секретное слово из кошелька YooMoney (HTTP-уведомления) | Вебхук `api/yoomoney/callback` |
| `EMAIL_SERVER_HOST` | Хост SMTP сервера отправки почты (например, `smtp.yandex.ru`) | Отправка писем, `lib/mail.ts` |
| `EMAIL_SERVER_PORT` | Порт SMTP (обычно `465` для SSL или `587` для STARTTLS) | Отправка писем, `lib/mail.ts` |
| `EMAIL_SERVER_USER` | Логин от SMTP аккаунта | Отправка писем, `lib/mail.ts` |
| `EMAIL_SERVER_PASSWORD` | Пароль от SMTP аккаунта (пароль приложения) | Отправка писем, `lib/mail.ts` |
| `EMAIL_FROM` | Email отправителя (например, `info@64lines.ru`) | Отправка писем, `lib/mail.ts` |

> [!danger] Безопасность в YooMoney
> Если переменная `YOOMONEY_SECRET` не задана или указана неверно, платежи будут зависать в статусе `PENDING`, так как сервер не сможет подтвердить достоверность вебхука от платежной системы.

## ⚙️ Конфигурация Next.js (`next.config.mjs`)

Файл настроек Next.js определяет параметры сборки:
- **Разрешенные домены изображений**: для корректного отображения обложек курсов.
- **TypeScript & ESLint проверки**: игнорирование ошибок при сборке на хостинге (если включено).
- **Экспериментальные опции**: настройки оптимизации пакетов.

Расположение в коде: `next.config.mjs`
