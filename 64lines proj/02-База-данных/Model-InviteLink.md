# Модель InviteLink (Ссылка-Приглашение)
> Хранит токены приглашений для регистрации новых пользователей с определенной ролью (например, быстрая ссылка для регистрации нового Тренера или Ученика).

## Расположение в коде
`prisma/schema.prisma (model InviteLink)`

## Детали
- `id` (String, cuid, PK) — ID записи.
- `token` (String, UK) — Уникальный токен ссылки.
- `role` (Role) — Роль, которая запишется пользователю при регистрации.
- `teacherId` (String?) — ID тренера, к которому автоматически привяжется ученик при переходе.
- `used` (Boolean, Default false) — Была ли ссылка уже использована.
- `createdAt` (DateTime, now()) — Время создания.

## Связи
- Использует: [[Model-User]]
- Используется в: [[API-Invites]], [[Invite-token-page]]

## Заметки / особенности
Позволяет автоматизировать онбординг групп учеников в класс конкретного тренера.
