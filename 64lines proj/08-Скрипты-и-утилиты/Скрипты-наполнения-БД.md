# Скрипты наполнения и инициализации базы данных
> Инструменты для первоначального развертывания структуры таблиц (seed) и загрузки шахматных задач в базу данных.

## Файлы
- `scripts/seed.ts`
- `scripts/seed-puzzles.js` (и копия в корне `seed-puzzles.js`)
- `scripts/check-themes.js`

## Детали использования

### 1. Первоначальный сид базы (`seed.ts`)
Запускается через Prisma:
```bash
npx prisma db seed
```
Создает базовую запись глобальных настроек [[Model-Settings]] (цены на подписку и анализ) и загружает демонстрационный учебный курс с первыми модулями.

### 2. Импорт тактических задач (`seed-puzzles.js`)
Загружает базу шахматных задач из CSV/JSON файлов в таблицу [[Model-Puzzle]]. Каждая задача парсится, извлекается начальный FEN, последовательность правильных ходов и текстовые теги тем Lichess.
```bash
node scripts/seed-puzzles.js path/to/puzzles.json
```

### 3. Проверка тем задач (`check-themes.js`)
Парсит существующую таблицу Puzzle и выводит в консоль список уникальных шахматных тем с указанием количества задач по каждой теме (используется для настройки фильтров поиска в [[Puzzles]]).

## Связи

- Используется в: Процесс развертывания и обновления проекта

## Заметки / особенности
Скрипт `seed-puzzles.js` оптимизирован под пакетную вставку (Prisma `createMany`), чтобы избежать переполнения памяти Node.js при загрузке баз на 10 000+ задач.
