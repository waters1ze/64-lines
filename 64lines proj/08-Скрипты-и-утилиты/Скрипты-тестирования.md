# Скрипты тестирования и отладки
> Набор отладочных скриптов для изолированного тестирования платежей, шахматных правил, почты и сессий.

## Файлы
- `scripts/test-yoomoney-webhook.js` — Имитация оплаты YooMoney.
- `test-email.js` — Проверка отправки писем.
- `test_chess.js` — Тестирование правил ходов chess.js.
- `test-pgn-split.js` — Тестирование PGN-парсера.

## Детали использования

### 1. Симуляция платежа YooMoney (`test-yoomoney-webhook.js`)
Используется для проверки прохождения оплаты без траты реальных денег:
```bash
node scripts/test-yoomoney-webhook.js <PURCHASE_ID> [AMOUNT]
```
1. Считывает `YOOMONEY_SECRET` из корневого файла `.env`.
2. Генерирует корректную SHA1-подпись для переданного `PURCHASE_ID` (берется из таблицы [[Model-Purchase]] в статусе PENDING).
3. Делает POST запрос с имитацией параметров YooMoney на `http://localhost:3000/api/yoomoney/callback`.
4. Бэкенд верифицирует подпись и переводит заказ в APPROVED.

### 2. Тестирование шахматных ходов (`test_chess.js`, `test_move.js`)
Позволяют проверить корректность работы библиотеки `chess.js` с превращениями пешек, рокировками и взятиями на проходе перед загрузкой в интерактивную доску.
