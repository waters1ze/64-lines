# Скрипты администрирования (CLI инструменты)
> Набор вспомогательных JS-файлов в корне проекта для быстрого управления БД из консоли сервера без запуска браузера.

## Файлы
- `make_admin.js`
- `make-teacher.js`
- `delete-user.js`
- `list-users.js`
- `fix_db.js`
- `read-logs.js`

## Детали использования

### 1. Повышение роли пользователя до Администратора
```bash
node make_admin.js example@email.com
```
Ищет пользователя с указанным email в таблице [[Model-User]] и меняет его роль на `ADMIN`.

### 2. Назначение роли Тренера (Учителя)
```bash
node make-teacher.js example@email.com
```
Меняет роль пользователя на `TEACHER`.

### 3. Удаление пользователя
```bash
node delete-user.js example@email.com
```
Каскадно удаляет запись пользователя из БД. Удаляет все его привязанные домашние задания, диалоги чата и покупки.

### 4. Исправление связей в базе (`fix_db.js`)
Служит для исправления битых полей `teacherId` и удаления сиротских записей ДЗ после отладки.

## Связи
- Использует: [[Model-User]], `lib/db.ts`
- Используется в: Ручной запуск разработчиком в терминале (PowerShell/Bash)

## Заметки / особенности
Перед запуском любого CLI-скрипта убедитесь, что в файле `.env` прописана корректная переменная `DATABASE_URL`, иначе скрипты завершатся с ошибкой подключения к PostgreSQL.
