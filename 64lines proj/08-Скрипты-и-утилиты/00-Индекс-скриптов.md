# Индекс скриптов и утилит проекта "64 линии"

В проекте есть вспомогательные скрипты, расположенные в папке `scripts/` и в корневой папке проекта. Они служат для инициализации БД, миграций, тестирования и административных задач.

## 🛠️ Скрипты администрирования (в корне проекта)

| Файл | Назначение | Документация |
|---|---|---|
| `make_admin.js` | Назначает пользователю (по email) роль ADMIN в базе данных | [[Скрипты-администрирования]] |
| `make-teacher.js` | Назначает пользователю (по email) роль TEACHER в базе данных | [[Скрипты-администрирования]] |
| `delete-user.js` | Полностью удаляет учетную запись пользователя по email | [[Скрипты-администрирования]] |
| `list-users.js` | Выводит список всех зарегистрированных пользователей в консоль | [[Скрипты-администрирования]] |
| `fix_db.js` | Исправляет несоответствия или битые связи в БД после миграций | [[Скрипты-администрирования]] |
| `check_db.js` | Быстрая проверка подключения к PostgreSQL через Prisma | [[Скрипты-тестирования]] |
| `read-logs.js` | Скрипт для форматирования и чтения логов ошибок | [[Скрипты-администрирования]] |

| Файл | Назначение | Документация |

## 🧪 Тестовые и отладочные файлы (в корне проекта)
В корне проекта лежит набор мелких `test-*.js` файлов для ручного запуска и проверки работы бэкенда:
- `test-pgn.js`, `test-pgn-split.js`, `test-multi-pgn.js` — проверка корректности парсинга шахматных вариантов.
- `test-email.js` — отладка SMTP-отправок.
- `test-session-student.js`, `test-session-all.js`, `test-latest-session.js` — отладка сессий живых уроков.
- `test_chess.js`, `test_move.js`, `test_promo.js`, `test_rapid.js` — изолированные тесты библиотеки `chess.js`.
- Подробный разбор отладочных скриптов доступен в [[Скрипты-тестирования]].
