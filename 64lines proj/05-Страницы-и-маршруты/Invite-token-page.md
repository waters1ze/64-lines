# Страница Принятия Приглашения (Invite Token Page)
> Динамический маршрут для обработки реферальных инвайт-ссылок, созданных тренерами или администрацией.

## Расположение в коде
`app/invite/[token]/page.tsx`

## Детали
- Извлекает параметр `[token]` из пути URL.
- Проверяет существование и статус токена приглашения в таблице [[Model-InviteLink]].
- Если токен валиден и не использован, предлагает пользователю быстро войти или зарегистрироваться для применения настроек приглашения.
- При регистрации по этой ссылке ученик автоматически привязывается к тренеру `teacherId`, указанному в инвайте.

## Связи
- Использует: [[Model-InviteLink]], [[API-Invites]], [[Model-User]]
- Используется в: Маршрут `/invite/[token]`

## Заметки / особенности
После успешной привязки статус инвайта меняется на `used = true`, предотвращая повторное использование одноразовых ссылок.
