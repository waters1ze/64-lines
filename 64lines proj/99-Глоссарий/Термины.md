# Глоссарий терминов проекта "64 линии"

В этом словаре собраны основные шахматные и технические понятия, используемые в исходном коде и документации проекта.

---

## ♟️ Шахматные термины

### FEN (Forsyth-Edwards Notation)
**FEN** — это стандартная текстовая нотация для описания конкретной шахматной позиции на доске. Она описывает положение всех фигур, чей сейчас ход, возможность рокировки, взятия на проходе и счетчик полуходов.
*Пример*: `rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1` (начальная позиция).
*Используется в*: [[Model-Puzzle]] (поле `fen`), [[Model-LiveSession]] (поле `currentFen`), [[Puzzles]] (отрисовка доски).

### PGN (Portable Game Notation)
**PGN** — текстовый формат для записи целых шахматных партий, ходов, разветвлений (вариантов) и текстовых комментариев.
*Используется в*: [[Model-Homework]] (поле `pgn`), [[Model-Opening]] (поле `pgn`), [[teacher-hub|PgnBoard]] (отрисовка дерева ходов).

### SAN (Standard Algebraic Notation)
**SAN** — стандартная алгебраическая шахматная нотация для записи ходов (например, `1. e4 e5 2. Nf3 Nc6`). Отличается от UCI (где ход пишется как координаты клеток, например `e2e4`).

### Elo рейтинг
**Elo** — метод расчета относительной силы игроков. Применяется в системе тактических задач школы: при верном решении рейтинг игрока растет, при ошибке — снижается. Сложность самой задачи также оценивается в Elo.
*Используется в*: [[Model-User]] (поле `rating`), [[Model-Puzzle]] (поле `rating`), [[Решение-задач-тактика]] (формула расчета).

## 💻 Технические термины

### Webhook (Вебхук)
**Вебхук** — механизм оповещения сервером платежной системы стороннего сервера (нашего) о событиях в реальном времени. Используется для автозачисления оплат.
*Используется в*: [[YooMoney]], [[API-Payments-and-YooMoney]].

### Long Polling / Polling
**Опрос сервера** — регулярная отправка AJAX/fetch запросов клиентом на сервер для получения свежих данных (например, раз в 2 секунды). Заменяет WebSockets.
*Используется в*: [[ChatComponent]] (проверка новых сообщений), [[LiveLessonBoard]] (проверка ходов тренера).

### Jitsi Room
**Видеокомната Jitsi** — изолированный зашифрованный канал связи для видеоконференций.
*Используется в*: [[Jitsi-живые-уроки]], [[LiveLessonBoard]].
