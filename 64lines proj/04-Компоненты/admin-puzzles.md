# Компонент AdminPuzzles (Загрузчик задач)
> Форма для администратора, позволяющая быстро добавлять новые тактические задачи в общую базу данных платформы.

## Расположение в коде
`components/admin-puzzles.tsx`

## Детали
### Функционал:
- Поля ввода: FEN начальной позиции, Строка правильных ходов (SAN/UCI), Рейтинг сложности (Elo), Темы (themes, разделенные пробелом).
- Интегрированная мини-доска для визуальной проверки вводимого FEN.
- Валидация: проверяет, корректен ли формат FEN и совпадают ли ходы решения с правилами шахмат (через `chess.js`).

## Связи
- Использует: [[API-Puzzles]] (api/puzzles/create)
- Используется в: [[teacher-hub]] (раздел add_puzzle для ADMIN)

## Заметки / особенности
Поддерживает пакетный импорт задач через вставку JSON-массива.
