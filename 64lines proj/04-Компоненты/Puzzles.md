# Компонент Puzzles (Тренажер тактики)
> Обеспечивает интерактивный интерфейс для решения тактических задач. Загружает задачу, отображает доску, проверяет ходы пользователя и отправляет результат на сервер.

## Расположение в коде
`components/Puzzles.tsx`

## Детали
### Особенности реализации:
- Использует `react-chessboard` для отображения доски и `chess.js` для контроля шахматных правил.
- **Логика решения**: Пользователю показывается начальная FEN позиция. Фигуры соперника делают первый ход из списка `moves`. Пользователь должен сделать правильный ответный ход. При правильном ходе доска воспроизводит следующий ход соперника, пока массив `moves` не будет исчерпан.
- При ошибке выводится сообщение "Неверный ход", доска сбрасывается к началу задачи.
- При полном правильном решении отправляется запрос на `/api/puzzles/submit` с флагом `isCorrect: true`.
- Принимает сложность (Легко, Средне, Сложно) и темы (Мат в 1 ход, Вилка, Связка и т.д.) через выпадающие списки.
