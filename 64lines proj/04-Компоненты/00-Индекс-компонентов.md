# Индекс клиентских React компонентов проекта "64 линии"

В каталоге `components/` расположены ключевые интерактивные элементы интерфейса приложения.

| Компонент | Расположение | Назначение | Связанные API / Модели | Документация |
|---|---|---|---|---|
| **TeacherHub** | `components/teacher-hub.tsx` | Ядро личного кабинета (ученик/учитель/админ) | Почти все API | [[teacher-hub]] |
| **Puzzles** | `components/Puzzles.tsx` | Тренажер тактических задач с Lichess | `/api/puzzles`, `Puzzle` | [[Puzzles]] |
| **LiveLessonBoard** | `components/LiveLessonBoard.tsx` | Синхронизированная интерактивная доска живого урока | `/api/live`, `LiveSession` | [[LiveLessonBoard]] |
| **ResizableBoard** | `components/ResizableBoard.tsx` | Контейнер изменения размера шахматной доски | нет (чистый UI) | [[ResizableBoard]] |
| **InviteComponents** | `components/InviteComponents.tsx` | Панели приглашений учеников и связки с тренером | `/api/invite`, `/api/students` | [[InviteComponents]] |
| **AdminPuzzles** | `components/admin-puzzles.tsx` | Панель загрузки кастомных задач для админа | `/api/puzzles/create` | [[admin-puzzles]] |
| **ChatComponent** | `components/ChatComponent.tsx` | Боковая панель чата и обмена сообщениями | `/api/chat`, `Message` | [[ChatComponent]] |
| **NotificationBanner**| `components/NotificationBanner.tsx` | Плашка системных уведомлений вверху экрана | `/api/notifications` | [[NotificationBanner]] |
| **Providers** | `components/providers.tsx` | Обертка контекстов NextAuth и стилей | NextAuth | [[providers]] |

## 🎨 Стилизация и UI-библиотеки
- В проекте используется **Tailwind CSS v4** для описания стилей.
- Базовая кнопка вынесена в UI-библиотеку: `components/ui/button.tsx`.
